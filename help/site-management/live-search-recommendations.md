# [!DNL Live Search] para recomendaciones

{{ee-feature}}

[!DNL Live Search] de Adobe Commerce ofrece una experiencia de búsqueda rápida, relevante e intuitiva para Adobe Commerce sin cargo adicional. [!DNL Live Search] con tecnología de Adobe AI usa inteligencia artificial y algoritmos de aprendizaje automático para realizar un análisis profundo de los datos agregados de visitantes. Estos datos, cuando se combinan con su catálogo de Adobe Commerce, resultan en experiencias de compra muy atractivas, relevantes y personalizadas. Este vídeo explica cómo utilizar la búsqueda en directo para potenciar las recomendaciones de productos y formaba parte de una serie de seminarios web en curso: &quot;Commerce quick wins&quot;.
