# Open telemetry

Descubra cómo la Telemetría Abierta estandariza la observabilidad a través de registros, seguimientos y métricas sin bloquearle en una sola herramienta de monitorización. En este vídeo se explica por qué la adopción de un estándar abierto para todo el sector permite una correlación perfecta de la señal. Learn how Open Telemetry supports existing stacks, and delivers end-to-end operational insights for a unified monitoring strategy.

## ¿Para quién es este vídeo?

* Adobe Commerce Developer
* Arquitecto de soluciones
* Gerente técnico de productos

## Contenido de vídeo

* Estandarizar la observabilidad con OpenTelemetry en todas las señales
* Correlacionar registros, seguimientos y métricas para un análisis de causas raíz más rápido
* Mantener las herramientas de monitorización existentes mientras se unifican los flujos de datos
