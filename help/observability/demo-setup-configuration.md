# Instalación y configuración

Aprenda a lograr la observabilidad de extremo a extremo en Adobe Commerce y aplicaciones personalizadas mediante telemetría estandarizada y correlacionada. En este vídeo se explica por qué son importantes la correlación y la estandarización, y se presenta el módulo Commerce Observability y la biblioteca de telemetría en directo de E/S. Descubra cómo estas herramientas permiten la monitorización, el registro y el seguimiento unificados para un flujo operativo fluido.

## ¿Para quién es este vídeo?

* Adobe Commerce Developer
* Arquitecto de soluciones
* Gerente técnico de productos

## Contenido de vídeo

* Estandarización y correlación de la telemetría para una observabilidad completa
* Uso del módulo Commerce Observability para señales automatizadas
* Ampliar la monitorización con la biblioteca de telemetría en directo de IO para aplicaciones
