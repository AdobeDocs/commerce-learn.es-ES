# Mejora del rendimiento de las tiendas con Edge Delivery

Descubra cómo las tiendas Adobe Commerce con tecnología Edge Delivery Services logran una velocidad y una resiliencia excepcionales. Descubra cómo el procesamiento inteligente de páginas, el almacenamiento en caché persistente y la arquitectura de CDN mejoran las puntuaciones de Lighthouse, Core Web Vitals y la SEO de forma predeterminada.
