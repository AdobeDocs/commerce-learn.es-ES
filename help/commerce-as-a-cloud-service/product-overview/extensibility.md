# Ampliación de Adobe Commerce con App Builder y API Mesh

Descubra cómo Adobe Commerce as a Cloud Service admite la extensibilidad a través de App Builder y API Mesh. Aprenda a integrar sistemas back-end, automatizar flujos de trabajo y crear aplicaciones personalizadas sin interrumpir la experiencia de la tienda.

## ¿Para quién es este vídeo?

* Arquitecto de soluciones
* Desarrollador back-end
* Especialista en integración

## Contenido de vídeo

* Descubra cómo App Builder permite el desarrollo de aplicaciones personalizadas y la automatización del flujo de trabajo para Adobe Commerce.
* Descubra cómo API Mesh organiza los datos en varios sistemas a través de una capa de API unificada.
