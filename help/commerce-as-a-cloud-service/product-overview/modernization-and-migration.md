# Migrar a Adobe Commerce as a Cloud Service

Descubra cómo Adobe Commerce proporciona estrategias de migración flexibles para las empresas que se trasladan a Adobe Commerce as a Cloud Service. Obtenga información sobre migraciones completas, adopción incremental y soluciones complementarias como Adobe Commerce Optimizer, todas diseñadas para reducir las interrupciones y acelerar el tiempo de respuesta al valor.

## ¿Para quién es este vídeo?

* Administrador de comercio electrónico
* Arquitecto de soluciones
* Administrador de proyectos de TI

## Contenido de vídeo

* Obtenga información acerca de las opciones de migración, incluida la migración completa, la adopción por fases y las soluciones complementarias.
* Descubra cómo Adobe admite la migración con herramientas, asistencia de socios y extensibilidad a través de App Builder y API Mesh.
