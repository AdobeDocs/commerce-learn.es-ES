# Añadir un módulo de JavaScript

{{only-for-on-prem-commerce-cloud}}

Aprenda a desarrollar un módulo JavaScript simple que muestre el saludo &quot;Hello World&quot; en la tienda.

## ¿Para quién es este vídeo?

* Desarrolladores

## Pasos

1. Cree un módulo.
2. Agregar `requirejs-config.js` y un archivo de módulo de JavaScript.
3. Cree una actualización de diseño que cargue una plantilla para el módulo.
4. Cree el archivo de plantilla.
5. Registre el módulo y pruébelo en la tienda.

## Contenido de vídeo

* Configure el módulo y Requiera la configuración de JS.
* Conecte el script mediante una actualización de diseño y una plantilla.
* Compruebe el saludo en la página.
